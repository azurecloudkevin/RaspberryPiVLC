# RaspberryPiVLC
This repository configures a Raspberry Pi device to operate as a demo host.  The following features are supported by code in this repo:
1. Crontab-compatible script to support a screensaver on reboot
2. Raspberry Pi GPIO input buttons
3. Raspberry Pi GPIO output buttons
4. Trigger of VLC video play by button push
5. LED strip control
6. Raspberry Pi inter-process-communication
